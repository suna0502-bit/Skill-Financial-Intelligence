💡 產品核心定位 (Core Value)
在多人理財與個人資產管理中，人們常面臨「多人代墊算不清」、「無效的轉帳冗餘路徑（A轉給B、B再轉給C）」與「記帳流程過於繁瑣」的痛點。
本系統為此而生，透過自動化計算法則與極簡的操作流程，將個人記帳、分帳群組與存錢目標完美整合，並確保所有資金流向具備高度的可追溯性與100%精準度。

🚀 三大核心模組特色 (Three Core Modules)
個人帳本模組 (Personal Ledger | 智能指令記帳)

智能命令解析 (NLP-like Command)：告別繁瑣的下拉選單！只需在智慧控制列輸入 記帳 早餐 150，系統即自動分類為飲食並寫入支出。
分類佔比分析：使用原生 Canvas 渲染發光 Donut 圓餅圖，直觀展現個人月度與年度收支佔比。
多人分帳模組 (Group Splitting | 債務極致簡化)

多樣化分攤模式：完美支援「平均分攤 (AA)」、「指定比例分攤」與「指定成員自訂金額」。
除不盡極端值自動平衡：當遇到金額除不盡時，小數點誤差自動由代墊者或最後一位成員吸收，保證全體成員分攤總和完美等於支出，帳目平衡率高達 100%。
存錢與儲蓄模組 (Savings Goal | 療癒進度反饋)

視覺化儲蓄卡片：即時計算並反饋儲蓄計畫的目前進度、達成率（%）以及距離目標的剩餘金額與天數。
帳本連動扣款：支持手動存入，或勾選「與個人帳本連動扣款」，存錢的同時自動在個人帳本上記入一筆「儲蓄」分類的支出，免除重複記帳。
💻 四大技術亮點與演算法創新 (Technical Innovations)
分分計數 (Cents-Based Integer Math) | 0 誤差承諾 系統所有底層與資料庫金額儲存均以「分 (Cents，即金額 ×100)」作為整數儲存，在 API 輸出時再自動還原成兩位小數。完全避開了計算機科學中浮點數運算（Floating-point errors）造成的累計誤差，確保系統容錯率為 0。

債務簡化動態規劃 (Subset Sum Zero-Sum DP) | 最少轉帳次數 為了解決多人共同旅行、合租中多方代墊的交互欠款，本系統導入了子集和動態規劃優化 (Subset Sum DP) 與貪婪演算法的雙重機制。系統能精確識別所有「可以互相抵銷且和為 0 的獨立子群組」，將轉帳筆數降到理論上的絕對最低限度（轉帳筆數 T≤N−1），自動消滅 A轉給B、B轉給C的冗餘清帳路徑。

工業級壓力測試 (QC Integration) 內置壓力測試控制台，能一鍵在後端生成 10 位虛擬測試員、寫入 55 筆以上複雜的混合分攤共同交易。演算法可在 1.3 毫秒 (1.3 ms) 內完成全自動結算並通過「零和平衡」與「最少轉帳次數」檢驗。

奢華視覺美學 (Premium UX Design) 網頁介面採用深色玻璃擬物風 (Dark Glassmorphism) 打造，融入粒子流光背景、物理拋物線落幣動效、卡片回彈抖動等精緻微表情，讓使用者在冰冷的財務數字中，體驗到手作設計的溫度。
